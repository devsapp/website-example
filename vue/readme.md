## 使用流程

<usedetail id="flushContent">

### 查看部署的应用
本项目案例是 Vue 部署到阿里云 Serverless 平台（函数计算 FC），部署完成之后，您可以看到系统返回给您的案例地址。

此时，打开案例地址，就可以进入 Vue 默认的首页。

### 二次开发

您可以通过云端控制台的开发功能进行二次开发。如果您之前是在本地创建的项目案例，也可以在本地项目目录`website-vue-v3`文件夹下，对项目进行二次开发。开发完成后，可以通过`s deploy`进行快速部署。

</usedetail>
